# Actividad-1---Tema-2


Este repositorio reúne los 10 ejercicios prácticos desarrollados durante la Actividad 1 del Tema 2 de Programación Web. Cada ejercicio tiene como objetivo reforzar los conocimientos fundamentales del desarrollo de páginas web utilizando HTML, CSS y JavaScript.

El proyecto cuenta con un archivo index.html, el cual funciona como un menú principal que permite acceder fácilmente a cada uno de los ejercicios.

Los ejercicios fueron elaborados para practicar la creación de páginas web mediante el uso de diferentes elementos de HTML, como encabezados, párrafos, listas, tablas, formularios, imágenes, enlaces y otros componentes que permiten estructurar correctamente el contenido de una página.

Además, algunos ejercicios incorporan estilos y funcionalidades para mejorar tanto la apariencia como la interacción con el usuario.

Se utilizaron hojas de estilo CSS (Cascading Style Sheets) para separar el diseño de la estructura del documento HTML.

Con CSS fue posible:

Modificar colores, fuentes y tamaños de texto.
Organizar y distribuir los elementos de la página.
Agregar márgenes, rellenos y bordes.
Personalizar botones, tablas y formularios.
Mejorar la presentación visual y la experiencia del usuario.

El uso de archivos CSS externos facilita el mantenimiento del código y permite reutilizar los mismos estilos en varias páginas del proyecto.

Se empleó JavaScript para añadir interactividad y comportamiento dinámico a las páginas.

Entre las funciones implementadas se encuentran:

Validación de datos ingresados por el usuario.
Respuesta a eventos como clics y cambios en formularios.
Realización de cálculos y conversiones.
Actualización del contenido de la página sin modificar el HTML.
Generación de mensajes y resultados de forma dinámica.

JavaScript complementa a HTML y CSS al permitir que las páginas web respondan a las acciones del usuario y ejecuten lógica de programación.
